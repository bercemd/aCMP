# aCMP
## Molecular dynamics simulations of amphiphilic collagen-mimetic peptides (aCMPs)
This repository contains data for the simulations of amphiphilic collagen-mimetic peptides (aCMPs).

The ```input/system_preperation``` folder contains the files for preparing the initial pdb files. 
To prepare the initial C<sub>12</sub>333 aCMP peptide, ```build.inp``` file is used as follows:
```
$CHARMMEXEC < build.inp > build.out
```
To prepare the concentrated aCMP systems, ```prep_system_rotate.py``` as follows:
```
python prep_system_rotate.py --input_path=model_ccbuilder_prl.pdb --output_path=triple_helix_12.pdb --cutoff=5 --box=100 --particle_number=4
```
To prepare the hexagonal assembly systems of aCMP, ```prep_system_assembly.py``` and ```prep_system_assembly_stacking.py``` scripts are used.

The ```input/system_preperation``` folder also contains the initial structure of C<sub>12</sub>333 aCMP peptide generated by CHARMM IC tables, initial triple helix generated by CCBuilder 2.0 and concentrated systems of single peptides and triple helices by the provided python scripts. 


The ```input/topology``` folder contains the topology file for lipidated proline residue. This residue topology was included into the ```toppar_all36_lipid_prot.str``` in the original CHARMM c36 topology file.


The ```analysis``` folder contains the Python scripts for the analysis of clustering, contact, rmsd, number of salt bridges and dihedral angles.

Examples of the usage of each analysis script are below:

CLUSTERING:
```
python clustering.py --title="copy12" --psf="copy12.psf" --selection="not name H*" --distance=2.7 --dcd=prod12.dcd
```

CONTACT:
```
python contacts_multiseg_mda.py --traj_path=prod12.dcd --psf_path=copy12.psf --selseg="P010=P020=P030=P040=P050=P060=P070=P080=P090=P100=P110=P120" --out_path=contact12.dat
```
